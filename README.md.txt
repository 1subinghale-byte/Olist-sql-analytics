# Olist SQL Analytics Project

This repository contains an end-to-end SQL Server analytics project
built on the Brazilian Olist e-commerce dataset.

## Scope
- Dimensional model (Facts & Dimensions)
- Data quality and sanity checks
- Business-focused analytical queries

## Tech Stack
- SQL Server
- T-SQL (window functions, CTEs, aggregations)

## Folder Structure
- `01_schema` – table & schema creation
- `02_staging` – raw ingestion logic
- `03_transform` – dimensional transformations
- `04_quality_checks` – data validation
- `05_analysis` – business analytics queries

## Key Business Questions
- Late delivery % by seller
- Rolling revenue trends
- Customer order behavior
